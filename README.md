A simple School Management System using:
- OOP (Student, Course classes)
- File Handling (data saved in students.txt)
- Encapsulation (private password with getter)

## Features
1. Enroll new student (saves to file)
2. Show all students (reads from file)
3. Search student by roll
4. Data persists after program restarts
